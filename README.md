## Projects

**InterviewForge** — [GitHub Repo](#) | [Live Demo](#)

Most interview prep tools just throw random questions at you. I wanted something smarter — so I built InterviewForge, an AI-powered platform that simulates real technical interviews based on your target role and difficulty level.

- Integrated an AI API to dynamically generate relevant interview questions
- Built a full auth system using JWT with protected routes and middleware
- Tracks each interview session and generates a performance report so users can see where they're improving
- Handled the full stack — React frontend, Node/Express backend, MongoDB for storing sessions and user data

This project taught me a lot about building secure, stateful applications and working with AI APIs in a real product.

**Stack:** React · Node.js · Express.js · MongoDB · JWT


**Air Quality Prediction (ML)** — [GitHub Repo](#)

For my machine learning project, I picked a real-world problem — predicting the Air Quality Index from environmental sensor data. Instead of just training one model and calling it done, I compared four different algorithms: SVR, Decision Tree, ExtraTrees, and XGBoost.

- Applied feature engineering to improve model input quality
- Tuned hyperparameters across models to reduce overfitting
- XGBoost gave the best results and generalized well on unseen data
- Documented results to make it easy to understand which model worked and why

This project gave me solid experience in the end-to-end ML workflow — data cleaning, modelling, evaluation, and iteration.

**Stack:** Python · Scikit-learn · XGBoost · Pandas · Feature Engineering
